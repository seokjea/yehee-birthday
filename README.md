# 예희 생일 축하 Streamlit 사이트

## 파일 구성

- `app.py` : 웹사이트 전체 코드
- `birthday_song.mp3` : Suno AI에서 만든 노래 파일
- `requirements.txt` : 필요한 패키지

## 실행 방법

1. 이 폴더 안에 Suno에서 만든 음악을 `birthday_song.mp3`라는 이름으로 저장합니다.
2. 터미널에서 현재 폴더로 이동합니다.
3. 아래 명령어를 실행합니다.

```bash
pip install -r requirements.txt
streamlit run app.py
```

4. 브라우저에서 사이트가 자동으로 열립니다.

## 참고

브라우저의 자동 재생 정책 때문에 페이지가 열린 직후 음악이 막힐 수 있습니다.
이 사이트는 첫 화면에서 사용자가 `편지 열기 · 노래 듣기` 버튼을 누른 뒤 음악이 표시되도록 구성되어 있습니다.
음악이 자동으로 시작되지 않으면 오디오 플레이어의 재생 버튼을 한 번 눌러주세요.

## 문구 수정

`app.py` 안의 다음 부분을 수정하면 됩니다.

- 첫 화면 제목
- 축제에서 처음 손을 잡았던 추억
- 생일 편지 내용
- 마지막 서명
